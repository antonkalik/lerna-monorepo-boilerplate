Lerna / Vite / Storybook - Monorepo
===================================

This is a monorepo template for a project using Lerna, Vite and Storybook.

## Getting Started

### Install dependencies

```bash
yarn install
```

### Run Storybook

```bash
yarn start:vite-common
```

### Build Storybook

```bash
yarn build:vite-common
```
or build all
```bash
yarn build
```

### Run tests

```bash
yarn test
```

### Run lint

```bash
yarn lint
```
